# 🐦 NOA-SVM：基于星鸦优化算法的支持向量机

## 1. 项目简介

本项目探索和实现了一种新型的**分类算法——NOA-SVM**（Nutcracker Optimization Algorithm - Support Vector Machine）。

传统的支持向量机（SVM）在处理分类问题时，其性能高度依赖于**软间隔惩罚参数 $C$** 和**核函数参数 $\gamma$** 的人工设置。同时，经典SVM缺乏对不同特征重要性的内在调整机制。针对这些挑战，本项目创新性地将**星鸦优化算法**引入SVM中。

NOA-SVM利用NOA的强大全局搜索和局部开发能力，同步地优化以下两类关键参数：

1.  **SVM核心参数：** $\gamma$ 和 $C$。
2.  **特征加权系数：** 赋予不同特征不同的重要性，提升模型的拟合和泛化能力。

通过这种结合，NOA-SVM能够自动寻找最优参数组合和特征权重，显著提升了模型的分类精度、稳定性和鲁棒性。

## 2. 项目特色与创新 ✨

### 💡 核心创新点

本项目最大的特色在于新兴的**星鸦优化算法（NOA）**与SVM结合，成功构建了一个**自适应参数和特征加权**的优化框架。

* **参数自适应调整：** 告别传统耗时且经验依赖的网格搜索，NOA在高维搜索空间（维度为 $m+2$，其中 $m$ 为特征数，2为 $C$ 和 $\gamma$）中高效探索最优解。
* **引入特征加权机制：** 针对不同特征对分类边界的贡献，NOA为每个特征分配权重，打破了传统SVM所有特征一视同仁的限制，使模型能够更关注关键特征，提高分类质量。

### ⚙️ 技术简述

本项目在技术上融合了多个模块：

1.  **基础分类器 (SVM)：** 采用了**径向基核函数（RBF核）**处理非线性问题，并实现了**软间隔**机制增强模型的鲁棒性。
2.  **优化求解器 (SMO)：** 使用**序列最小优化（SMO）算法**高效求解SVM对偶问题中的拉格朗日乘子 $\alpha$。
3.  **元启发式算法 (NOA)：** NOA模拟星鸦四季中的**搜寻、储存、检索**食物行为，包含探索和开发算子，用于在搜索空间中迭代更新（优化）$\gamma$、$C$ 和特征权重系数。
4.  **适应度评估：** 将NOA生成的参数和权重输入SVM进行训练和验证，以验证集上的**分类准确率的倒数**作为个体的**适应度值**，指导NOA寻找全局最优解。
5.  **多分类策略 (OVO)：** 针对多分类任务（如Iris数据集），采用 **One-vs-One (OVO)** 策略，对每两个类别训练一个独立的二分类器，最终通过**投票**决定样本类别。

## 4. 系统架构

| 文件/目录名 | 描述 |
| :--- | :--- |
| `src/main.py` | 🚀 **项目主入口**：负责加载数据集（Iris或乳腺癌），执行模型的运行和评估。 |
| `src/nutcracker.py` | 🐦 **星鸦优化算法**：封装了 NOA 的完整逻辑。它是一个**通用的优化器**，接收评估函数 $f$ 进行寻优。 |
| `src/svm.py` | ⚙️ **SVM核心与集成**：包含基础**二分类 SVM** 实现（RBF 核函数，Platt SMO 训练）和**多分类器**。多分类器实现了基于 NOA 优化的 OVO 策略。 |
| `src/DecisionTree.py` | 🌳 **决策树算法**：实现了基础的决策树分类器。使用**信息增益**（基于**熵**的计算）来确定最佳分割点。|
| `src/evaluate.py` | 📊 **实验评估模块**：负责数据预处理、数据集划分，以及模型的评估，并包含绘制**混淆矩阵**和 **ROC 曲线**的功能。 |
| `docs/` | 📝 **实验报告文档：** 存放完整的实验报告、详细数据表格和所有结果图示。 |

## 5. 使用说明

本项目代码结构完整，模型在运行 `main.py` 时会自动进行训练和优化。

1.  **下载代码：** 请确保您已完整下载本项目的全部代码文件。
2.  **启动程序：** 直接运行主文件 `main.py` 即可开始 NOA-SVM 的优化和训练过程。

```bash
python main.py
```
